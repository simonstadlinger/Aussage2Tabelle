# Aussage2Tabelle
Erstellt eine Latex-Tabelle aus einer Aussage

## Ausführen
Zum Ausführen einfach die Datei `a2w`.py mit python3 laufen.

## Eingabe
Die Eingabe folgende Kriterien erfüllen:   
- Erlaubte Zeichen: ~ + - > = ( ) a-z   
- Und: +
- Negation: ~
- Oder: -
- Implikation: >
- Äquivalenz: =   

Beispiel: `a - ~b + (b = c)`
